![nix][icon-48] nix
===

Sonification of http browser session traffic.

## Installation

1. Visit [about:debugging](about:debugging)
2. Click `Load Temporary Add-on..`
3. Select `*/nix/manifest.json`

## Development

Setup and development:
```
cd nix/
yarn
yarn watch
```
